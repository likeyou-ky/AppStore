# Django-RentBuddy-Website
*This is a RentBuddy Website using Django and Heroku-postgressql*

* Find people of similar interest
* Trending activities
* Give ratings (buddy and client)
* Settings Page (Update interest, personal particulars)

* JavaScript
## Back-end
