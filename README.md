# Django-RentBuddy-Website
*This is a RentBuddy Website using Django and Heroku-postgresql*

# Features
* Find people of similar interest
* Trending activities
* Give ratings (buddy and client)
* Buddy update page
* Client update page

* CSS
* Bootstrap
* JavaScript

## Back-end
* Django
* Postgresql
